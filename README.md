### REACT CRUD LIBRARY APP

### ABOUT

This is a React Library App that performs CRUD operations. It can Create New
Books, Read Stored Books, Update the books and Delete books from storage in
reactjs using react hooks api.

### TECHNOLOGIES USED:

    * HTML5
    * CSS3
    * Bootstrap4
    * Reactjs and Apis

### FEATURES

    * Create Books
    * Update Books
    * Read Data
    * Delete Books
